## 📝 Marketing Descriptions

### Short Description (Max 80 characters)
A clean, modern note-taking app with a premium interface for daily productivity.

### Long Description
**Easy Note** is your minimalist personal companion for capturing ideas, tasks, and inspirations instantly. Designed with a premium aesthetic and high-performance focus, it helps you stay organized without the clutter.

**Key Features:**
- **Modern Masonry Layout:** See more of your notes at a glance with a beautiful, dynamic grid.
- **Instant Search:** Find any note in seconds with our optimized search engine.
- **Premium Design:** A clean Blue & White theme designed for focus and readability.
- **Offline First:** Your data stays on your device using local storage for maximum privacy and speed.
- **Lightweight & Fast:** Built with Jetpack Compose for a smooth 60fps experience.

## 🚀 Quick Start Guide (for Developers)

### Phase 1: Clone the Project 📥
```bash
git clone https://github.com/sonph1812/easyNote.git
```

### Phase 2: Open and Sync ⚙️
1. **Open Android Studio**: Launch the IDE.
2. **Open Project**: Select **File > Open** and choose the `easyNote` folder.
3. **Gradle Sync**: Wait for the message *"Gradle sync finished"* at the bottom of the screen.

### Phase 3: Run the App 📱
1. Select your device or emulator.
2. Click the green **Run** button (or press `Shift + F10`).
